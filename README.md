1. Устанавливаем зависимости

        composer install

2. Генерируем ключ

        php artisan key:generate

3. Разворачиваем миграции

        php artisan migrate

4. Генерируем ключи для аутентификации

       php artisan passport:keys

5. Добавляем в таблицу роли
      
       php artisan db:seed
